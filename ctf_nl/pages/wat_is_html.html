Wat is HTML

HTML, oftewel HyperText Markup Language is een soort script die bepaalt wat de inhoud en structuur van een website is.

Het bestaat uit tags en elementen. Een tag geeft de stijl of functie van iets aan, terwijl een element juist een benoemde onderdeel van de HTML bestand is, zoals een sectie.

Bijvoorbeeld: om tekst dikgedrukt te maken moet het geplaatst worden tussen de "<strong>" tags. Als tags geopend worden moeten ze ook gesloten worden, met "</strong>" tag, in ons geval.
