Wat is CSS?

CSS, oftewel Cascading Style Sheets, is een scripting taal dat het ontwerp van een webpagina bepaalt. De font-stijl, achtergrond kleur, font-grootte, etc. worden allemaal bepaald door CSS. Je kunt de CSS van een website zien via de Web Developer Tools (F12) door te kiezen voor de Style Editor tool.



Terwijl je daar bent, kun je de CSS zelfs aanpassen!
