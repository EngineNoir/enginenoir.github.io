Wat is JavaScript?

JavaScript is een programmeertaal gebruikt voor web ontwerp. De meeste interacties op een gegeven website worden uitgevoerd met behulp van JavaScript. Hieronder kun je op een knopje drukken die de vlag zal printen in de Netwerk Log. Om de log te zien moet je Web Developer Tools (F12) gebruiken en de console tab kiezen.



JavaScript is heel breed en veelgebruikt. Om er verder in te duiken, kun je meer leren en oefenen hier.
